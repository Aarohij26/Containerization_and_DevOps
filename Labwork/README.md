## 🧪 Lab Experiments

The table below provides an overview of the experiments included in this repository:

| Experiment No. | Title | Result |
|:--------------:|:------|:-------|
| [Experiment 1](./Experiment1/README.md) | VM vs Container – Utilization | Demonstrated that containers are more lightweight and resource-efficient compared to virtual machines |
| [Experiment 2](./Experiment2/README.md) | Docker Installation, Configuration, and Running Images | Docker images were successfully pulled, containers executed, and lifecycle commands performed |
| [Experiment 3](./Experiment3/README.md) | Deploy NGINX Using Different Base Images & Compare Layers | Deployed NGINX with different base images and compared image size and layers |
| [Experiment 4](./Experiment4/README.md) | Docker Essentials: Dockerfile, Tagging and Publishing | Built custom Docker images using Dockerfile, tagged them, and pushed to Docker Hub |
| [Experiment 5](./Experiment5/README.md) | Docker Volumes, Environment Variables, Monitoring & Networks | Implemented persistent storage, environment configurations, container monitoring, and networking between containers |
| [Experiment 6](./Experiment6/README.md) | Docker Compose vs Docker Run – Multi-Container WordPress + MySQL | Successfully deployed a multi-container application using Docker Compose and compared it with docker run |
| [Experiment 7](./Experiment7/README.md) | CI/CD using Jenkins, GitHub and Docker Hub | Implemented a CI/CD pipeline to automate build and deployment using Jenkins |
| [Experiment 9](./Experiment9/README.md) | Ansible Configuration Management | Automated configuration and deployment tasks using Ansible playbooks |
| [Experiment 10](./Experiment10/README.md) | SonarQube Code Quality Analysis | Analyzed code quality and detected bugs, vulnerabilities using SonarQube |
| [Experiment 11](./Experiment11/README.md) | Container Orchestration using Docker Swarm | Deployed and managed services using Docker Swarm for orchestration |
| [Experiment 12](./Experiment12/README.md) | Kubernetes Container Orchestration | Implemented container orchestration using Kubernetes and analyzed its advantages over Docker Swarm |
